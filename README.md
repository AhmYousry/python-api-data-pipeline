# Python API Data Pipeline

This project demonstrates a simple data pipeline built with Python.

The pipeline collects data from a public API, processes it, and exports the results into a CSV file.

## Features

• Fetches data from REST APIs
• Processes JSON data
• Converts data into structured format
• Exports results to CSV

## Technologies

* Python
* Requests
* Pandas

## Workflow

API → Data Processing → CSV Export

## Usage

Install dependencies:

pip install -r requirements.txt

Run the pipeline:

python main.py
